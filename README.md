### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、快递员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告
5. 货物
6. 留言板
7. 网点信息
8. 个人中心
9. 收货地址
10. 货物订单


**快递员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 公告管理
6. 订单分配管理
7. 留言板管理
8. 网点信息管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 快递员管理
5. 用户管理
6. 公告管理
7. 货物管理
8. 货物订单管理
9. 订单分配管理
10. 留言板管理
11. 网点信息管理
12. 基础数据管理（公告类型管理、货物类型管理、订单类型管理、网点类型管理）
13. 轮播图管理
